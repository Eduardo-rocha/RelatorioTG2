\chapter{Descrição do Conteúdo do DVD}\label{AnCD}

O DVD entregue possui uma cópia digital do relatório e arquivos relevantes para o desenvolvimento do trabalho. O conteúdo completo é:
\begin{itemize}
\item Cópia digital do relatório e seu projeto {\LaTeX}.
\item Desenhos 2D e 3D das peças feitos no SolidWorks.
\item Dados coletados durante os testes.
\item Diagramas elétricos do sistema.
\item Fotos e vídeos da máquina em funcionamento.
\item Projeto de programação do CLP completo.
\end{itemize}
